# HealthAPP 应用介绍

## 项目概述

HealthAPP 是一款专为健康监测设计的移动应用，与 ESP32S3 智能手表配合使用，实现实时健康数据的采集、可视化和分析。应用支持心率监测、步数统计，并具备异常心率检测、设备断连提醒等功能，为用户提供全面的健康管理体验。

## 核心功能

### 1. 健康数据监测
- **心率监测**：实时采集并显示心率数据，支持历史数据趋势分析
- **步数统计**：记录每日步数，提供运动数据可视化
- **异常心率检测**：当心率过高时自动提醒用户

### 2. 设备管理
- **BLE设备连接**：通过蓝牙与ESP32S3智能手表配对
- **设备状态监测**：实时检测设备连接状态，断连时自动提醒
- **数据同步**：与手表设备实时同步健康数据

### 3. 用户体验
- **实时图表**：动态展示心率和步数数据变化趋势
- **异常提醒**：当检测到异常心率时，通过弹窗、通知和震动提醒用户
- **个性化设置**：支持个人信息管理和设备设置

## 技术栈

### 前端技术
- **Flutter**：跨平台移动应用开发框架
- **Provider**：状态管理库
- **Flutter Local Notifications**：本地通知功能
- **HapticFeedback**：设备震动反馈

### 后端技术
- **ESP32S3**：智能手表硬件平台
- **BLE（低功耗蓝牙）**：设备通信协议

### 开发工具
- **Android Studio**：应用开发环境
- **Visual Studio Code**：代码编辑器
- **Gradle**：Android构建工具

## 安装与运行

### 前置条件
- **Flutter SDK**：2.0.0 或更高版本
- **Android Studio**：4.0 或更高版本
- **ESP32S3智能手表**：已烧录对应固件

### 安装步骤
1. **克隆项目**
   ```bash
   git clone https://github.com/yourusername/HealthAPP.git
   cd HealthAPP
   ```

2. **安装依赖**
   ```bash
   flutter pub get
   ```

3. **运行应用**
   ```bash
   flutter run
   ```

### 设备连接
1. 确保ESP32S3智能手表已开机
2. 在应用中进入"设备连接"页面
3. 搜索并选择对应的ESP32S3设备
4. 等待连接成功提示

## 功能详解

### 异常心率检测
1. **触发方式**：在心率详情页面点击"异常心率检测"按钮
2. **检测逻辑**：
   - 随机生成120-180的心率值
   - 将异常心率绘制到图表中并以醒颜色标注
   - 弹出"心率过高异常，请降低运动强度"的提示
   - 发送系统通知
   - 触发设备震动（5秒震动，1秒暂停，循环3次）
3. **用户操作**：
   - 点击"我知道了"按钮：停止震动，关闭提示
   - 不点击：继续震动循环

### 设备断连检测
- 实时监测BLE设备连接状态
- 断连时自动显示提醒信息
- 在心率和步数页面均有断连检测功能

### 图表展示
- **心率图表**：
  - Y轴标签位于左侧
  - 自动调整最大值以适应数据范围
  - 异常数据以醒颜色标注
- **步数图表**：
  - 每日步数趋势展示
  - 直观的柱状图显示
